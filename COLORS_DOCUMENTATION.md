## ملف المصدر

جميع هذه الألوان معرّفة في:
```
lib/presentation/theme/app_style.dart
```

---

## مثال على الاستخدام في الكود

```dart
import 'package:foodyman/presentation/theme/app_style.dart';

// استخدام اللون الرئيسي
Container(
  color: AppStyle.primary,
)

// استخدام لون النص
Text(
  'مرحبا',
  style: TextStyle(color: AppStyle.textGrey),
)

// استخدام لون الخلفية
Scaffold(
  backgroundColor: AppStyle.mainBack,
)
```
